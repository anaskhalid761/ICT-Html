History of HTML

An overview of HTML’s evolution, important versions, features, and best practices for accessible, semantic markup.

Timeline

  • 1991 — HTML 1.0: Tim Berners-Lee publishes the first HTML spec; basic tags like <p>, <a>, <img>.
  • 1995 — HTML 2.0 / 3.2: Forms and tables standardized; broader browser adoption.
  • 1999 — HTML4: Separation of content and presentation encouraged; accessibility awareness grows.
  • 2000s — XHTML era: XML-based stricter parsing model adopted by some projects.
  • 2014 — HTML5: Semantic elements (<article>, <section>), multimedia (<video>, <audio>), APIs (Canvas, Web Storage).
  • 2016 — present — Living standard: WHATWG publishes continuous updates; modern APIs and accessibility improvements.

Key features by version

  • Semantic markup: <header>, <nav>, <main>, <footer>
  • Forms & validation: new inputs (email, date), constraint validation
  • Multimedia: native <video> and <audio>
  • APIs: Canvas, Web Storage, Service Workers, Fetch
  • Accessibility: ARIA roles and focus management best practices

Code example

Minimal semantic page header:

<header>
        <h1>Site title</h1>
        <nav><a href="#">Home</a></nav>
</header>
